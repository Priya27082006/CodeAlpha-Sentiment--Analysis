# CodeAlpha - Task 4: Sentiment Analysis

## Project Overview
This project performs Sentiment Analysis on Amazon product reviews using Python. It classifies reviews into Positive, Negative, and Neutral categories using the TextBlob library.

## Features
- Load Amazon Reviews Dataset
- Analyze Review Sentiments
- Classify Reviews as Positive, Negative, or Neutral
- Generate Bar Chart
- Generate Pie Chart
- Save Results to CSV
- Display Final Insights

## Technologies Used
- Python
- Pandas
- Matplotlib
- TextBlob
- NLTK

## Files Included
- amazon_reviews.csv
- sentiment_analysis.py
- sentiment_results.csv (Generated after running the program)
- requirements.txt
- README.md

## How to Run
1. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```

2. Run the program:
   ```
   python sentiment_analysis.py
   ```

## Output
- Sentiment Classification
- Sentiment Summary
- Bar Chart
- Pie Chart
- Generated CSV file (sentiment_results.csv)
