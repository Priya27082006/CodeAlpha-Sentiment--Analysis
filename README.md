# CodeAlpha - Task 4: Sentiment Analysis

## Project Overview
This project performs Sentiment Analysis on Amazon product reviews using Python and the TextBlob library. It classifies customer reviews into Positive, Negative, and Neutral sentiments and visualizes the results using charts.

## Objectives
- Analyze text data.
- Classify reviews as Positive, Negative, or Neutral.
- Visualize sentiment distribution.
- Generate insights from customer opinions.

## Technologies Used
- Python
- Pandas
- Matplotlib
- TextBlob
- NLTK

## Project Structure
```
Task4_Sentiment_Analysis/
│── amazon_reviews.csv
│── sentiment_analysis.py
│── requirements.txt
│── README.md
│── sentiment_results.csv (Generated after execution)
```

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python sentiment_analysis.py
```

## Output
- Displays the dataset information.
- Classifies reviews as Positive, Negative, or Neutral.
- Shows sentiment summary.
- Generates a Bar Chart.
- Generates a Pie Chart.
- Saves the results in `sentiment_results.csv`.

## Conclusion
This project demonstrates how Natural Language Processing (NLP) can be used to analyze customer reviews and identify overall sentiment.
